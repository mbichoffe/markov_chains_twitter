About

This project is a take-home assignment that was incredibly fun to build. It is a Markov Chain generator that takes in the tweets of a Twitter user and generates a random text based on those tweets. The generated text can then be tweeted if the user is logged in to her Twitter account(via OAuth).

In this project I've used Tweepy, an open-source and easy-to-use library that allows your Python programming projects to access the Twitter API, but there are many other high-quality Python wrappers for Twitter out there.

The random text is generated through the creation of n-grams. By analyzing the n-grams and listing which words most often follow a given n-gram, we can construct new text that sounds like the source text.

I was inspired by much of what I've learned at Hackbright Academy and by some incredible repos like Markovify.

This project is built using python and flask, and there is still a lot to be done. If you're interested in contributing to this project, please feel free to fork the repository and make some changes. I'm still a newbie, so please be nice :) I'll make sure to give you credit here once you send your pull request.
